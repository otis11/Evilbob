FAQ

What is Bob?

Bob is a command palette for your browser. It aims to provide all interfaces of a browser, like bookmarks, history and more advanced features through a command palette.

How to install Bob?

Install Bob as an extension for your browser. Links ^

How to open Bob?

Bob uses browser keyboard shortcuts to open the command palette.
The default shortcut is
CtrlShiftL
If the shortcut is not working, set the shortcut yourself.

Chrome

  1. Type chrome://extensions/shortcuts into your search bar.
  2. Set the shortcut for "Open Bob"

Firefox

  1. Type about:addons into your search bar.
  2. Click on the right cog wheel
  3. Click on "Manage Extension Shortcuts"
  4. Set the shortcut for "Open Bob"

Edge

  1. Type edge://extensions/shortcuts into your search bar.
  2. Set the shortcut for "Open Bob"

What are prefixes?

Prefixes are used to show results for 1 plugin only. They are the first few characters typed in the search field. Type ? to list all available prefixes. Prefixes are only available if the corresponding plugin is enabled. For instance, use h Test to search for "Test" in your browsing history, when the History plugin is enabled.

How to change Settings?

  1. Open Bob
  2. Search for "bob Settings" and select
  3. Change your settings. They will be saved automatically.

How to enable/disable Plugins?

  1. Open Bob
  2. Search for "bob Open Plugins" and select
  3. Click on any plugin to disable or enable it

How to navigate results?

Press
ArrowDown
to select the next result or change the keybinding in Bob Settings.
Press
ArrowUp
to select the previous result or change the keybinding in Bob Settings.

How to select a result?

Press
Enter
or change the keybinding in Bob Settings.

How to show result options?

Press
ShiftEnter
or change the keybinding in Bob Settings.
